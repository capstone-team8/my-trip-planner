# my-trip-planner

## Project setup
### Install dependencies
Run this command on the **root** folder, the **server** folder, and the **client** folder.
```
npm install
```

### Run the project
#### Run the server on dev mode
```
npm run server
```

#### Run the client on dev mode
```
npm run client
```

#### Run the server and the client concurrently
```
npm run dev
```
